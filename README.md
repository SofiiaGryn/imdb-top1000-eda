# IMDb Movies Analysis — Sprint 3 (Turing College)
## Overview
This project was completed as part of Sprint 3 at Turing College. It focuses on analyzing the IMDb Top 1000 Movies dataset from Kaggle.
The goal is to explore the data, clean it, and identify patterns between different variables such as ratings, revenues, genres, and release decades.

### Objectives

1) Clean and preprocess the dataset by removing duplicates, handling missing values, and renaming columns where necessary.
2) Conduct a guided exploratory data analysis (EDA) using Pandas and Matplotlib.
3) Interpret findings and highlight the most relevant insights.

## Dataset

The dataset contains 1,000 IMDb movies with the following key columns:

Title, Director, Genre

IMDB_Rating, Meta_score, No_of_Votes

Revenue (millions), Runtime (minutes), Release_Year

The films span from the 1920s to the 2020s, covering various genres and production styles.
